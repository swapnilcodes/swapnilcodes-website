Frontend Vs Backend

Frontend Vs Backend A website consists of two sides -:

1. client-side

2. server side

The client-side is called frontend and server-side is called backend. The Frontend is all About view and design of the website whereareas, the backend is logic-building and the working of the website. A devloper who devlops the frontend and design of the website is called a frontend-developer, the basic requirements for a frontend devloper are given below:

languages-:

1. html (Hyper Text Markup Language) for basic structure of the website

2. css (cascade style sheet) for styling the website

3. js (javascript) for working of the website eg. Event handling, media queries etc.
frameworks (optional)-:

Reactjs, Angularjs etc.

and the devloper who builds the backend of the website is called a backend developer, the basic requirements for a backend devloper are given below-:

A backend framework eg. Django(python), node.js(javascript), flutter(java), flask(python) etc.

A database eg. Mongodb, Mysql etc.

And a fullstack devloper is the one who can work with both frontend and backend...
